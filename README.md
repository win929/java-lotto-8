# java-lotto-precourse

### 1. 로또 구입
- [ ] 로또 구입 금액을 입력받는다.
    - [ ] 입력값이 숫자가 아닌 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.
    - [ ] 입력값이 1,000원 단위가 아닌 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.
    - [ ] 입력값이 0 이하인 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.

### 2. 로또 발행
- [ ] 구입 금액에 해당하는 개수만큼 로또를 발행한다.
    - [ ] 1장의 로또는 1~45 사이의 중복되지 않는 6개 숫자로 생성한다.
- [ ] 발행된 로또 번호를 `Lotto` 클래스로 관리한다.
    - [ ] `Lotto` 클래스 유효성 검증
        - [ ] 로또 번호에 중복된 숫자가 있는 경우 `[ERROR]` 예외 처리
        - [ ] 로또 번호가 1~45 범위 밖인 경우 `[ERROR]` 예외 처리

### 3. 발행 로또 출력
- [ ] 발행한 로또 수량을 출력한다.
- [ ] 발행한 모든 로또의 번호를 오름차순으로 정렬하여 출력한다.

### 4. 당첨 번호 입력
- [ ] 당첨 번호 6개를 쉼표(`,`) 기준으로 입력받는다.
    - [ ] 쉼표(`,`) 외의 구분자를 사용하거나 숫자/쉼표 외 문자가 포함된 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.
    - [ ] 숫자가 6개가 아닌 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.
    - [ ] 1~45 범위 밖의 숫자가 포함된 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.
    - [ ] 중복된 숫자가 포함된 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.

### 5. 보너스 번호 입력
- [ ] 보너스 번호 1개를 입력받는다.
    - [ ] 입력값이 숫자가 아닌 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.
    - [ ] 1~45 범위 밖의 숫자인 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.
    - [ ] 이미 입력된 당첨 번호 6개 중 하나와 중복되는 경우 `[ERROR]` 메시지 출력 후 다시 입력받는다.

### 6. 당첨 통계 계산
- [ ] 발행한 로또와 당첨 번호를 비교하여 당첨 내역을 계산한다.
- [ ] Java Enum을 활용하여 당첨 등수(Rank)를 관리한다.
- [ ] 각 등수별 당첨 횟수를 집계한다.
- [ ] 총 당첨금을 계산한다.
- [ ] 총 수익률을 계산한다. (수익률은 소수점 둘째 자리에서 반올림)

### 7. 결과 출력
- [ ] 당첨 통계 결과를 요구사항의 출력 형식에 맞게 출력한다.
- [ ] 총 수익률을 요구사항의 출력 형식에 맞게 출력한다.
